# FitnessApp
Фитнес приложение для создания фигуры своей мечты.

## Functionalities
- Пользователь каждый день выполняет определенное количество упражнений на время с перерывами на отдых.



## Technologies
- droidsonroids.gif
- viewBinding
- RecuclerView
- Fragments
